# GMMK v2 白猫 (White cat)

My personal configuration for the GMMK v2 65%. Fairly simple for now, just some lighting and media keys.


## Features

- Caps lock indicator
- Per-layer lighting
- 1000hz, 2ms debounce, 12 events per scan
- VIA support
